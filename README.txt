CONSIGNA PROYECTO:

Elaborar un sitio web, preferiblemente un e-commerce, en donde se puedan aplicar los contenidos vistos en el curso:

- Sintaxis / variables
- Ciclos / iteraciones
- Funciones
- Objetos
- Arrays
- Funciones de orden superior
- DOM
- Eventos
- Storage/JSON
- Operadores avanzados
- Librerías
- Ajax & Fetch




PRESENTACIÓN:

Plataforma:
Store de video juegos


Flujo para el usuario:

1) Sección HOME.
2) Seleccionar video juegos (añadirlos al carrito).
3) Sección CARRITO.
4) Sumar/restar cantidades de producto, eliminar producto por completo.
5) Comprar.
6) Pop up de confirmación.
7) Repetir el ciclo.


FUNCIONES RELEVANTES:

- Timer.
- Creación de cards dinámicas.
- Creación de chips para valoraciones: distintos colores y distinta imagen (estrella feliz/triste) según la puntuación.
- Toggle favoritos.
- Contador para productos agregados al carrito.
- Agregar/eliminar productos al local storage.
- Creación de tabla dinámica para la lista de productos en carrito.
- Sumar/restar/eliminar productos en la tabla.
- Funciones de librerías para pop up y toast.


ACLARACIONES:
- La API solo me brinda los datos referentes a los juegos, menos el precio, por lo tanto el "precio anterior", el "precio final" y el "porcentaje de descuento" son 3 factores que se relacionan entre sí, pero son datos generados cada uno de manera aleatoria.
